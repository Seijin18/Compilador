Gerenciamento de Registradores Ineficiente
Descrição:

Registradores não são liberados após uso, causando "vazamento".

Não há salvamento de registradores callee-saved.

Solução:

Liberar registradores explicitamente após uso.

Salvar/restaurar registradores s0-s7 no prólogo/epílogo.

Adições ao Prólogo/Epílogo:
// No prólogo
for (int i = 16; i <= 23; i++) { // s0-s7
    add_addi_subi_instruction(SP, SP, -4);
    add_instruction("SW", OP_SW, SP, i, 0, 0, NULL);
}

// No epílogo
for (int i = 23; i >= 16; i--) {
    add_instruction("LW", OP_LW, SP, i, 0, 0, NULL);
    add_addi_subi_instruction(SP, SP, 4);
}

Operações com Imediatos Negativos
Descrição:

Instruções como ADDI/SUBI não tratam imediatos negativos corretamente.

Solução:

Usar SUBI para imediatos negativos e ADDI para positivos.

Código Corrigido (em add_addi_subi_instruction):
void add_addi_subi_instruction(int src_reg, int dst_reg, int immediate) {
    if (immediate >= 0) {
        add_instruction("ADDI", OP_ADDI, src_reg, dst_reg, 0, immediate, NULL);
    } else {
        add_instruction("SUBI", OP_SUBI, src_reg, dst_reg, 0, -immediate, NULL);
    }
}

Passagem de Parâmetros Frágil
Descrição:

Parâmetros além dos 4 primeiros não são passados corretamente (offsets calculados erroneamente).

Não há desalocação de parâmetros extras após a chamada.

Solução:

Passar parâmetros extras via pilha com SP.

Desalocar espaço após a chamada com ADDI SP, SP, N.

Código Corrigido (em generate_assembly_second_pass):
else if (strcmp(quad->op, "call") == 0) {
    // ... (código existente)
    for (int p = 0; p < param_count; p++) {
        if (p < 4) {
            // Passa em registradores a0-a3 (R4-R7)
        } else {
            int src_reg = load_variable_to_register(temp_params[p].name, current_function);
            // **Correção:** Passa parâmetro via SP
            add_addi_subi_instruction(SP, SP, -4);
            add_instruction("SW", OP_SW, SP, src_reg, 0, 0, NULL);
        }
    }
    add_instruction_with_label_fix("JAL", OP_JAL, 0, 0, 0, quad->arg1);
    // **Correção:** Desaloca parâmetros extras
    if (param_count > 4) {
        int extra_bytes = (param_count - 4) * 4;
        add_addi_subi_instruction(SP, SP, extra_bytes);
    }
    // ... (código existente)
}
