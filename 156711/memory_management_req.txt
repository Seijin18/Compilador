# Requisitos de Gerenciamento de Memória - Compilador 156711

## 📋 Objetivo
Este documento especifica os requisitos de gerenciamento de memória para o sistema de síntese do compilador 156711, detalhando como o assembler deve gerenciar registradores, pilha e escopo de variáveis.

## 🎯 Requisitos Implementados ✅

### 1. Gerenciamento de Registradores 
- **Algoritmo LRU** implementado para alocação inteligente
- **29 registradores de uso geral** (R1-R28, R29=SP)
- **Registradores especiais**: R29 (SP), R30 (FP), R31 (RA)
- **Sistema de spill/reload** para otimização

### 2. Gerenciamento de Pilha
- **Stack Pointer (R29)** para controle principal
- **Frame Pointer** para contexto de funções
- **Múltiplas pilhas** (principal + contexto)
- **Alinhamento automático** de frames

### 3. Escopo de Variáveis
- **Separação automática** entre variáveis locais e globais
- **Tabela de símbolos** com informação de escopo
- **Offsets automáticos** para posicionamento na pilha
- **Contexto de função** preservado em chamadas

### 4. Passagem de Parâmetros
- **Registradores preferenciais** para poucos parâmetros
- **Pilha principal** para múltiplos parâmetros
- **Ordem correta** de salvamento/restauração
- **Suporte a recursão** com contexto preservado

### Layout da Pilha (Activation Record)
```
+------------------+ ← SP atual
| Return Address   | (offset 2, 6, 8...)
+------------------+
| Parâmetros      | (offset 0, 1...)
+------------------+
| Variáveis Locais | (offsets crescentes)
+------------------+
| Temporários     | (t0, t1, t2...)
+------------------+ ← SP anterior
```
